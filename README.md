# Liquid Glass 精美玻璃态起始页
✨ 一款高颜值、多功能的玻璃态（Frosted Glass）浏览器起始页，支持亮色/暗色/壁纸三种模式，集成时钟、搜索、日期、天气、待办清单、快捷入口等实用功能，搭配炫酷打字机特效，视觉与体验兼备。

## 📸 效果预览
### 核心特性
- 🍃 液态玻璃视觉效果，适配**亮色/暗色/壁纸**三种显示模式，质感拉满
- 🕒 实时数字时钟+日期显示，精准同步系统时间，大字号清晰易读
- 🔍 智能搜索功能，支持**网址直跳+关键词搜索**，操作便捷
- 📋 本地存储待办清单，刷新页面数据不丢失，轻量无负担
- ⚡ 自定义快捷入口，一键直达常用网站，效率翻倍
- 🎬 专属打字机推广特效，逐字输入+删除动画，播完支持重播
- 📱 全响应式设计，完美适配电脑/平板/手机等全尺寸设备
- 💬 豆包快捷入口专属泡泡框emoji图标，视觉识别度高

📋 详细功能说明

一、显示模式切换

支持3种模式自由切换，模式偏好自动本地记忆，刷新不重置

• ✨亮色模式：清爽白底黑字，视觉干净，适合白天办公学习

• 🌙暗色模式：护眼黑底白字，降低屏幕反光，适合夜间使用

• 🖼️壁纸模式：模糊背景+通透玻璃效果，颜值天花板，氛围感拉满

二、核心工具组件

1. 时钟&日期 🕒📅

• 时钟：实时显示 时:分:秒，字体大小自适应屏幕，无挤压变形

• 日期：同步展示年-月-日 + 星期，信息完整，一目了然

2. 天气组件 ☁️

• 模拟天气数据展示，包含温度、天气描述、天气emoji图标

• 可自行对接天气API，替换为实时天气数据（自定义教程见下文）

3. 待办清单 ✅

• 输入内容按回车快速添加待办事项

• 点击文字可标记/取消「完成」状态（划线效果）

• 点击右侧×可删除待办，操作简单

• 所有数据通过localStorage本地存储，无需后端，隐私安全

4. 智能搜索 🔍

• 网址直跳：输入网址（如baidu.com），自动识别并跳转对应网站

• 关键词搜索：输入内容按回车/点击搜索按钮，调用必应搜索引擎

• 搜索框自带玻璃态悬浮/聚焦动效，交互反馈清晰

三、快捷入口 ⚡

内置5个常用快捷入口，卡片式设计，悬浮有上浮+背景变色动效：

1. 🏠 我的主站 - 跳转个人主页

2. 🎮 游戏厅 - 直达游戏页面

3. 💻 GitHub - 跳转个人GitHub仓库

4. 📚 必应 - 快速打开必应搜索

5. 💬 豆包对话 - 专属泡泡框图标，一键进入豆包聊天界面

🔧 自定义修改指南

1. 修改快捷入口

找到代码中shortcut-group区域，修改对应标签属性即可：
<a href="你的目标网址" class="shortcut-card">
    <div class="shortcut-icon">自定义emoji图标</div>
    <div class="shortcut-text">入口名称</div>
</a>
2. 修改打字机文字内容

找到JS代码中typingTexts数组，自由增删/修改文字：
const typingTexts = [
    '第一段文字',
    '第二段文字',
    '自定义推广文案',
    // 可无限增删
];
3. 更换壁纸背景

找到CSS中.dynamic-bg的background属性，替换图片链接：
.dynamic-bg {
    background: url('你的壁纸图片链接') no-repeat center center;
}
4. 修改默认显示模式

找到JS代码中initMode函数，修改默认模式（可选：light/dark/wallpaper）：
const savedMode = localStorage.getItem('liquidMode') || 'wallpaper';
5. 对接实时天气API

将天气组件的模拟数据，替换为第三方天气API接口返回值，即可实现实时天气显示。
